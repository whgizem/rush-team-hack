## 5. Enable GitHub Pages

Settings → Pages → Source: **Deploy from branch** → `main` → `/ (root)`

This gives you a second indexed URL:
`https://[yourusername].github.io/rush-team-hack`

## 6. Pin a discussion / create first issue
