## Troubleshooting

**Panel doesn't open?**
Make sure Tampermonkey is enabled and the script is active. Check the Tampermonkey icon in your browser toolbar — it should show a number badge.

**Script not loading?**
Reinstall from [rushteam.tools](https://rushteam.tools). The script auto-updates but sometimes a fresh install is needed.

**Rush Team Hack not working after game update?**
The script updates automatically. If you're on an old cached version, go to Tampermonkey dashboard → find the script → click "Check for updates".
